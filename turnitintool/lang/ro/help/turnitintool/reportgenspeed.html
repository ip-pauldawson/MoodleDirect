Teme Turnitin - Viteza generării raportului

Există trei opțiuni pentru această setare a temei: "Generați rapoartele imediat (redepunerile nu sunt permise)", "Generați rapoartele imediat (redepunerile sunt permise până la data limită de depunere)" și "Generați rapoartele la data limită de depunere (redepunerile sunt permise până la data limită de depunere)".

Opțiunea "primul raport este final imediat" generează Raportul privind originalitatea imediat când un student face o depunere. Când această opțiune este selectată studenții dumneavoastră nu vor putea să redepună la temă.

Pentru a permite redepuneri, selectați opțiunea "imediat (poate înlocui rapoartele până la data limită de depunere)". Aceasta permite studenților să redepună lucrările la temă în mod continuu până la data limită de depunere. Este posibil să dureze până la 24 de ore pentru a procesa Rapoartele privind originalitatea pentru redepuneri.

Opțiunea "la data limită de depunere" va genera un Raport privind originalitatea doar la data limită de depunere a temei. Această setare va face astfel încât toate lucrările depuse la temă vor fi comparate una cu cealaltă când sunt create Rapoartele privind originalitatea.
