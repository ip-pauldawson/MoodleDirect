Trabalhos Turnitin - Velocidade de Geração do Relatório

Há três opções para essa configuração de trabalho: "Gerar relatórios imediatamente (os reenvios não são permitidos)","Gerar relatórios imediatamente (os reenvios são permitidos até a data de entrega)" e "Gerar relatórios na data de entrega (os reenvios são permitidos até a data de entrega).

A opção "imediatamente o primeiro relatório é o final" gera o relatório de originalidade imediatamente quando um aluno faz o envio. Com essa opção selecionada, seus alunos não poderão reenviar o documento para o trabalho.

Para permitir os reenvios, selecione a opção "imediatamente (pode substituir os relatórios até a data da entrega)". Isso permite que os alunos reenviem os documentos ao trabalho continuamente até a data de entrega. Pode levar até 24 horas para processar os relatórios de originalidade dos reenvios.

A opção "na data de entrega" irá gerar o relatório de originalidade somente na data de entrega do trabalho. Essa configuração fará com que todos os documentos enviados ao trabalho serão comparados entre si quando os relatórios de originalidade forem criados.
